# Imágenes para las apps

Esta carpeta concentra recursos gráficos reutilizables por múltiples aplicaciones.

## Archivos disponibles

- `logo.png`: versión principal del logo.
- `logo2.png`: variante alternativa del logo.

## Recomendaciones de uso

- Preferir rutas relativas (`img/logo.png`) para facilitar portabilidad entre entornos.
- Mantener proporción de aspecto al redimensionar.
- Evitar reexportar múltiples copias del mismo recurso con nombres ambiguos.

## Convención para nuevas imágenes

Usar nombres claros y consistentes, por ejemplo:

- `logo-principal.png`
- `logo-horizontal.png`
- `icono-analitica.png`

## Optimización

Antes de incorporar nuevas imágenes:

1. Comprimir sin pérdida perceptible.
2. Verificar peso final (idealmente lo más bajo posible).
3. Validar visualización en fondos claros y oscuros, si aplica.
